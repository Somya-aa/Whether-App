# Weather App

A simple Weather App built using Python, Basic Flask, HTML, and the OpenWeather API. Users can enter a city name and view current weather information.

## Features

* Search weather by city
* Display temperature
* Display Feels Like
* Show humidity
* Show wind speed
* Display country

## Technologies Used

* Python
* Basic Flask
* HTML
* Requests
* OpenWeather API

## How to Run

1. Install required packages:

```bash
pip install flask requests
```

2. Add your OpenWeather API key:

```python
API_KEY = "YOUR_API_KEY"
```

3. Run the application:

```bash
python app.py
```

4. Open:

```text
http://127.0.0.1:5000
```
